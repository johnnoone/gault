# Tutorials

Welcome to the Gault tutorials. These guided lessons will teach you the fundamentals of working with Gault, a lightweight MongoDB ODM for Python.

Each tutorial is a hands-on walkthrough. Follow along from start to finish, and by the end you will have working code that you understand.

## Getting started

- [Your First Schema](first-schema.md) -- Define a schema, connect to MongoDB, insert documents, query them, and learn how persistence tracking works.
- [Building Aggregation Pipelines](building-pipelines.md) -- Construct aggregation pipelines to filter, group, and reshape data from a MongoDB collection.
